RoamCalibGUI
GitHub

A comprehensive camera calibration GUI developed at ROAM-Lab. The tool supports multiple calibration patterns and provides a full calibration workflow. Its core feature is a homography-based perspective correction pipeline: the user selects four points on a planar calibration target, and the application computes the homography matrix to rectify and present the target as a flat, undistorted plane — enabling accurate intrinsic parameter estimation.

The project exists in two versions, each refining the calibration interface and workflow based on research iteration.

Computer Vision Camera Calibration Homography C++ OpenCV Qt
ImgPersTool
v2 v1

A standalone image perspective correction tool, available in two versions. The user interactively selects four corner points on any planar surface within an image. The application then computes the homography transform between the selected quadrilateral and its rectified target plane, producing a flat, bird's-eye-view output. This makes it useful for document scanning, whiteboard capture, and image-based measurement tasks.

Computer Vision Homography Perspective Correction Python OpenCV
PAUGRAD
GitHub

A corner refinement and homography tool that leverages image gradient direction to improve the accuracy of user-selected points. The user selects approximate corner locations on a planar pattern; the algorithm then analyzes local gradient orientations to snap each selection to the true geometric corner. The refined corners are subsequently used to compute a homography transform, correcting perspective distortion with sub-pixel precision.

Computer Vision Corner Detection Gradient Analysis Homography Python OpenCV

Duygularımın Günlüğü — Mental Health Diary App
GitHub

A mental health journaling application designed for both therapists (experts) and their clients, running entirely on-device without any cloud dependency. The system uses Spring Boot for the local backend service and Flutter for the mobile interface. Clients can record daily emotional entries and reflections; therapists can review client journals and track progress over time. The fully offline architecture prioritizes user privacy and data sovereignty.
